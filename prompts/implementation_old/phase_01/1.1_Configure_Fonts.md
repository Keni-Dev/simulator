# Phase 1.1: Configure Fonts

**Phase:** 1 - Landing Page & Setup Screen
**Step:** 1.1

**Context:**
The core typography ensures the scientific feel of the simulator. 'Inter' is the primary sans-serif for UI labels, while 'JetBrains Mono' handles the high-precision data readouts.

**Skills to Load:**
- `frontend-design` — Applies the typography specs natively using Next/Font.
- `ui-ux-pro-max` — Handles exact variable setups for Tailwind.

**Instructions:**

1. **Modify `src/app/layout.tsx`:**
   - Import `Inter` and `JetBrains_Mono` from `next/font/google`.
   - Setup the variables:
     - `const inter = Inter({ subsets: ['latin'], variable: '--font-inter' })`
     - `const jetbrains = JetBrains_Mono({ subsets: ['latin'], variable: '--font-jetbrains' })`
   - Apply them to the `<body className={...}>`: Merge the variables string natively into the className array (`${inter.variable} ${jetbrains.variable}`).
   - *Ensure the Next.js `metadata` object is updated to "TriboSim - Conservation of Charge Simulator" instead of the default generic title.*

**Verification:**
- The DOM element `<body>` cleanly outputs both CSS variables mapped to standard classes.
- The Title updates.
