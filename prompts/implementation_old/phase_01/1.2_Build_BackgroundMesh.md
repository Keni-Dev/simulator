# Phase 1.2: Build the Background Mesh Component

**Phase:** 1 - Landing Page & Setup Screen
**Step:** 1.2

**Context:**
This element is pure UI/UX styling. Instead of a solid black wall, the background utilizes subtle CSS neon gradients simulating out-of-focus charge clouds (red and cyan), overlaying a pure black grid matrix.

**Skills to Load:**
- `ui-ux-pro-max` — Render gradient meshes via robust `blur()` filters and exact scaling.

**Instructions:**

1. **Create `src/components/ui/BackgroundMesh.tsx`:**
   - Scaffold a functional React component rendering an absolute, pointer-blocking underlayer.
   - Use: `<div className="fixed inset-0 pointer-events-none -z-10 bg-canvas-bg overflow-hidden">`
2. **Implement the CSS Grid Pattern:**
   - Render a child `div` containing a repeating seamless CSS background. You can achieve the "1px dot every 32px" look using a `radial-gradient` or pure CSS pattern pointing to `#2A2A2D` or generic `rgba(255,255,255, 0.05)`.
3. **Implement the Glowing Orbs:**
   - Create two absolutely positioned, rounded `div`s centered roughly in the background space.
   - Orb 1 (Negative Cloud): `bg-neon-cyan/5 blur-[100px] w-[600px] h-[600px] rounded-full mix-blend-screen`
   - Orb 2 (Positive Cloud): `bg-neon-red/4 blur-[100px] w-[500px] h-[500px] rounded-full mix-blend-screen`
   - Position these slightly offset from the exact center to simulate a dynamic space.

**Verification:**
- Importing and rendering `<BackgroundMesh />` any blank page immediately renders the dark grid with beautiful, smooth glowing spots.
