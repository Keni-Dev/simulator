# Phase 5.1: Aesthetic PBR Materials & Charge Auras

**Phase:** 5 - 3D Assets, VFX, and Polish
**Step:** 5.1

**Context:**
Takes the MVP from functional to visually coherent. We replace the spheres with geometries representing the materials (e.g., a cylinder for a glass rod) and apply high-quality React Three Fiber PBR (Physically Based Rendering) materials. The meshes also gain a glowing aura based on their charge.

**Skills to Load:**
- `3d-web-experience` — To configure PBR materials and emissive point lights mapped to Zustand physics state.

**Instructions:**

1. **Create `src/components/scene/materials/PhysicalMaterial.tsx`:**
   - Scaffold a client component receiving `hand: 'left' | 'right'`.
   - Read the current `materialId` and `q` from the Zustand store for that specific hand.
   - Look up the `Material` type from `MATERIALS`.

2. **Dynamic Geometry Selection:**
   - Write a helper switch statement based on the `MaterialType`.
   - If it's a rod (Glass/Teflon), render a `<Cylinder>` geometry (from `drei`).
   - If it's a fabric (Silk/Nylon), render a slightly squashed `<Box>` or an imported basic `.glb` geometry.

3. **PBR Material Definition:**
   - Wrap the geometry in a `<mesh>`.
   - Apply a `<meshStandardMaterial>` child.
   - Map standard properties: E.g., Copper (`metalness={1} roughness={0.1} color="#C07050"`), Glass (`transparent={true} opacity={0.6} roughness={0}`), Silk (`roughness={0.9} color="#FFFAFA"`).

4. **Dynamic Charge Aura (The "Glow"):**
   - Add a `<PointLight>` and/or modify the `emissive` and `emissiveIntensity` of the `<meshStandardMaterial>` based on `q`.
   - **Colors:** If `q > 0`, color is `#FF3D00` (Red/Positive). If `q < 0`, color is `#00E5FF` (Cyan/Negative).
   - **Intensity:** `Math.abs(q) * scalar`. Adjust `scalar` so it doesn't blow out the rendering engine immediately. Make sure it caps out at a reasonable max intensity.

**Verification:**
- The 3D scene now displays realistic-looking materials (e.g., a shiny metallic rod for copper) based on the HUD dropdown selection.
- As the user accumulates charge by rubbing, the material clearly glows with a colored aura scaling in intensity.
