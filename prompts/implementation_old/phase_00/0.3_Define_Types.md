# Phase 0.3: Define Shared Types

**Phase:** 0 - Project Scaffolding & CI/CD Setup
**Step:** 0.3

**Context:**
TriboSim's physics engine requires strict typing to prevent math and state errors in the continuous 60fps loop. This step defines the core data interfaces used throughout the application.

**Skills to Load:**
- `typescript-expert` — Implement strict typing for the physical simulation properties.

**Instructions:**

1. **Create `src/types/physics.ts`:**
   - Export a string literal union type: `export type MaterialType = 'insulator' | 'conductor';`
   - Define and export an interface `Material`:
     ```typescript
     export interface Material {
       id: string;
       name: string;
       type: MaterialType;
       triboIndex: number;
       description: string;
     }
     ```
   - Define and export an interface `ChargeState`:
     ```typescript
     export interface ChargeState {
       q1: number;
       q2: number;
       qNet: number;
     }
     ```

**Verification:**
- The file `src/types/physics.ts` exists and accurately exports the 3 definitions with zero TypeScript errors.
