# Phase 0.1: Initialize Next.js Project & Dependencies

**Phase:** 0 - Project Scaffolding & CI/CD Setup
**Step:** 0.1

**Context:**
This step establishes the foundational Next.js application, configures strict TypeScript styling, sets up Tailwind CSS, and prepares the core 3D and physics dependencies required for TriboSim.

**Skills to Load:**
Please apply these exact behaviors before coding:
- `clean-code` — Establish coding standards for maintainability.
- `cc-skill-coding-standards` — Apply strict TypeScript, React, and general conventions.
- `environment-setup-guide` — Standard Next.js/React project setup protocols.

**Instructions:**
1. Initialize a new Next.js 14+ application using `npx create-next-app@latest . --typescript --tailwind --eslint --app --src-dir --import-alias "@/*"`.
2. Install the core 3D rendering dependencies: `npm install three @react-three/fiber @react-three/drei lucide-react zustand`
3. Install the computer vision dependencies: `npm install @mediapipe/hands @mediapipe/camera_utils`
4. Install CSS/Tailwind utility libraries: `npm install clsx tailwind-merge`

**Verification:**
- `package.json` contains all the specified packages.
- A standard Next.js App Router directory structure (`src/app`) exists without errors.
