# Phase 0.2: Configure Global Styles and Tailwind

**Phase:** 0 - Project Scaffolding & CI/CD Setup
**Step:** 0.2

**Context:**
Applying the strict "Sci-Fi Lab" dark mode aesthetic. This sets the pitch black/charcoal background and disables native window scrolling to ensure the React Three Fiber (SPA) canvas fills the viewport seamlessly.

**Skills to Load:**
- `frontend-design` — Ensure high-contrast neon styling mapping to positive and negative charges.

**Instructions:**

1. **Modify `src/app/globals.css`:**
   - Override default Next.js global styling.
   - Force `html` and `body` tags to have `margin: 0`, `padding: 0`, `overflow: hidden`, `width: 100vw`, `height: 100vh`.
   - Set the body background color to deep charcoal `#0B0B0D` and global text color to `#FFFFFF`.

2. **Modify `tailwind.config.ts`:**
   - Extend the theme `colors` object with:
     - `neon-cyan`: `'#00E5FF'`
     - `neon-red`: `'#FF3D00'`
     - `deep-charcoal`: `'#0F0F12'`
     - `canvas-bg`: `'#0B0B0D'`
   - Extend the theme `fontFamily` object to map to Next.js CSS variables (we will import fonts in the layout later):
     - `sans`: `['var(--font-inter)']`
     - `mono`: `['var(--font-jetbrains)']`

**Verification:**
- `globals.css` correctly locks the viewport and defines the dark background.
- `tailwind.config.ts` contains the custom neon variables.
